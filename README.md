# Waste Classification Using Deep Learning

## A Computer Vision System for Automated Waste Sorting

**Research Type:** Computer Vision · Deep Learning · Image Classification

**Model:** EfficientNet-B0

**Tools:** Python · TensorFlow · Keras · Google Colab · Matplotlib

**Project Area:** Environmental Sustainability · Smart Waste Management · AgriTech

---

## Abstract

This project develops a deep learning-based waste classification system using EfficientNet-B0. The model classifies waste images into different categories to support automated sorting, cleaner environments, recycling decisions, and sustainable waste management.

The project demonstrates how computer vision can be applied to environmental and agricultural systems where waste segregation, composting, and recycling decisions can be improved through intelligent automation.
